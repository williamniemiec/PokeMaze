## üéÆ Controls
|        Command        |Description|
|----------------|-------------------------------|
| `w` | Move up |
| `a` | Move left |
| `s` | Move down |
| `d` | Move right |
| `p`| Perspective projection |
| `o`| Parallel projection |
| `c` | Free camera mode |
| `Pause Break` | Pause / Continue game |
| `Left mouse button` pressed + `direction`  | Moves player camera to `direction` |
